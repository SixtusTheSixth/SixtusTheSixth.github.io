Playlist Heardle

This is a variation of the game Heardle, which itself is a variation of the game Wordle. In the original, you have to guess the song title from the first few seconds of the song. You have six guesses, each of which gives you a little more of the song.

Here, you can upload a link to your own Spotify playlist, and you'll be quizzed only on songs from that playlist. I made this because I was frustrated by the song selections of the original Heardle websites (here or here). Credit for the idea also goes to my friend Owen Rollins. In my experience, this game is best played with friends, projected onto a shared screen.

Unfortunately, due to changes in Spotify's API terms, you need to have a Spotify Premium account to use this. You can create one here. Additionally, only user-created playlists will work. Spotify-owned or algorithmically created playlists (Blends, Daily Mixes, Liked Songs, etc.) are prohibited by the API.

Enter a Spotify playlist link (e.g. https://open.spotify.com/playlist/7B2M7EVfDdCMeKbj33dVYd for my whistling playlist):

  • 1. Open Spotify and find your desired playlist.
    2. Right-click the playlist (or open it and tap the "..." menu on mobile).
    3. Select "Share" then "Copy link to playlist".
    The link will look something like:
    "https://open.spotify.com/playlist/7N0gg0AJvQiVr5wCAeDaUw?si=fZhDaSYJST2WCcx3QVJBdw"
    4. Paste the link in the input box above.
